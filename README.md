Ropa-Carrito es una tienda virtual sencilla donde los usuarios pueden:
- Registrarse e iniciar sesión
- Agregar productos al carrito
- Ver descuentos
- Finalizar la compra
- Recibir un resumen de su compra por correo electrónico



Tecnologías usadas
- PHP
- Bootstrap 5
- PHPMailer
- MySQL (XAMPP)
